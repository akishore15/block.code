Block.code

Block.code is designed to be a replacement for Blockly, the JavaScript library to code in blocks.

Blockly Docs are here.

To get started programming in Block.code, go to the link below.

Steps to get Block.code:

  1. Go to the link above.
  2. Clone the repo.
  3. Open the Linux & Unix terminal.
  4. Type in mkdir blockcode.
  5. Run it.
  6. Type in cd blockcode.
  7. Run it.
  8. Type in git clone https://github.com/akishore15/block.code.
  9. Run it.
  10. Find the directory "blockcode" inside your file manager.
  11. Get the files.
  12. Open your text editor.
  13. Put in all the code.
  14. Run the code.

Voila! Your own version of Blockly, that you can use.

If you need more info on Blockly, watch this video.
